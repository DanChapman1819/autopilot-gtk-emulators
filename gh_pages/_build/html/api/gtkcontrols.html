This simply clicks a treeview object

select_item(instance, *args, **kwargs)[source]¶

Selects an item in a GtkTreeView by its UI label text

Parameters:label_text – The label value of the tree item as seen on the UI
Returns:An object of the requested treeitem

e.g. If you want to click say an item displaying ‘Home’ in a treeview then it would be:

>>> treeitem = treeview.select_item('Home')

also see GtkFileChooserDialog.go_to_directory() for another example

if for some reason this doesn’t work then use the .click() function to get the treeviews focus
